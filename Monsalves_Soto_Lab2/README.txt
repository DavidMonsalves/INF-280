Es necesario instalar las librerías "numpy", "matplotlib", "seaborn", "scipy", y "ipywidgets" las cuales se pueden instalar mediante los siguientes comandos:

>	pip install numpy
>	pip install matplotlib
>	pip install seaborn
>	pip install scipy
>	pip install ipywidgets
